Detailed Description

DriveBase for the Defense Drive Chassis.

Defense Drive chassis for Outlaw drive chassis. The alogrithem behind driving the chassis is unorthodox, require free body diagram analysis of the drive chassis. Due to this, refer to Allen for detailed explanation on drive chassis algorithem.

This is the defense drive's drive function, which controlls the chassis by direction, thrust, and turnfactor values.

Parameters
dir- The direction we drive towards, this is measured in radians.
thrust- This is how much power is to be given to the motors. The input value range is -1 to 1. If the value is zero, the motors won't be given any power. The thrust is calculated with vector math.
turn- This is the turning factor. Input range is -1 to 1, if zero, no turning will be done.
