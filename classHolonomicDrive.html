Detailed Description

DriveBase for the Holonomic Drive Chassis.

The Holonomic chassis is a chassis(obviously) with omniwheels for forward directions at 135 and 45 degrees. These chassis can move in all directions while turning in place. When turning the drive chassis, the chassis will sacrifice maximum forward force to create a moment about its center. Essentially, the faster you turn, the slower you accelerate.

Creates a Holonomic Drive with four generic speed controllers.

Refer to Desperado/Renegade documentation for use. Motor numbers start from the front right motor, and this one will called "motor1." The number process proceeds counter clockwise. So "motor2" will then be the front left motor. "motor3" will be the back left motor. And finally "motor4" will be the back right motor.

Parameters
motor1- front right motor
motor2- front left motor
motor3- back left motor
motor4- back right motor

This is the holonomic's drive function, which controlls the chassis by direction, thrust, and turnfactor values.

Parameters
dir- The direction we drive towards, this is measured in radians.
thrust- This is how much power is to be given to the motors. The input value range is -1 to 1. If the value is zero, the motors won't be given any power. The thrust is calculated with vector math.
turn- This is the turning factor. Input range is -1 to 1, if zero, no turning will be done.
