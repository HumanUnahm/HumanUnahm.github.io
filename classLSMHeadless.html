Detailed Description

This object handles communication and integration of gryodata.

This object is meant to interface with Pololu's MinIMU-9 v5 (Pololu #2738). It initializes communication and integrates gyro data to track relative angles about the z-axis.

int16_t LSMHeadless::getBias ( )

Returns gyro bias.

Gyro bias is associated with gyro drift. The larger the bias, the faster the gryo will drift. The bias value is calculated upon calling calling calibrate(), and is then compensated for when calling iterate().

Returns
Bias - Raw gryo value that represents bias.
